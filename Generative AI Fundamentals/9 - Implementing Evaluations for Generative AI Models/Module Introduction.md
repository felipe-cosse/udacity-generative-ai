How do you know if a generative AI model is any good? This module provides a practical toolkit for answering that critical question. You'll explore a range of essential evaluation techniques, starting with simple, strict methods like Exact Match. You will then move on to more nuanced approaches for text and code, such as lexical similarity with ROUGE, semantic similarity with embeddings, and functional correctness with unit tests. Finally, you’ll learn how to use another LLM as a judge for complex, subjective tasks.

What You'll Learn
Apply various metrics like Exact Match, ROUGE, and Semantic Similarity to evaluate text outputs.
Assess the functional correctness of generated code using unit tests and the Pass@k metric.
Implement the "LLM-as-a-Judge" pattern by creating a rubric to evaluate subjective model outputs.
Select the appropriate evaluation technique based on the specific Generative AI task.

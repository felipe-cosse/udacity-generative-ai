Welcome! In this module, you'll discover how to build powerful text classifiers using foundation models. You'll start with a sentiment classifier for movie reviews and then build a spam detector. Along the way, you'll master prompt engineering techniques like "zero-shot" and "few-shot" prompting to guide the model and improve its accuracy. You'll also learn to structure prompts to get reliable JSON output for easy evaluation.

What You'll Learn
Build text classifiers for tasks like sentiment analysis and spam detection.
Differentiate between and implement "zero-shot" and "few-shot" prompting techniques.
Craft effective system prompts to instruct a foundation model and request specific output formats like JSON.
Evaluate the performance of your classifier by calculating its accuracy.
Adapt a prompt-based approach to new classification problems.
